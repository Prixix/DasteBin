# DasteBin

DasteBin is a paste server that allows you to save code snippets to IPFS.

## Demo

You can view the demo at [https://dastebin.prixix.com](https://dastebin.prixix.com).

## Installation

    $ git clone https://github.com/prixix/dastebin.git

    $ cd DasteBin

    $ yarn install

    $ yarn start
